## Description

Profile badge for codeforces & atcoder & topcoder

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Using docker container
```bash
$ docker-compose up -d
```
